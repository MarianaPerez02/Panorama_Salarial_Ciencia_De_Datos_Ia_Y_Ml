# Panorama Salarial Ciencia De Datos Ia Y Ml

## Descripción general
Este proyecto presenta un análisis descriptivo y visual del panorama salarial en el ámbito de **Ciencia de Datos**, **Inteligencia Artificial (IA)** y **Machine Learning (ML)**.  
El análisis se desarrolla utilizando **Power BI**, con el objetivo de identificar patrones salariales, diferencias por rol, nivel de experiencia y tendencias relevantes del mercado laboral en estas áreas tecnológicas.

El archivo principal del proyecto es un reporte interactivo en formato `.pbix`, el cual permite explorar la información mediante visualizaciones dinámicas y filtros.

---

## Objetivo del proyecto
Analizar y visualizar la información salarial relacionada con perfiles de Ciencia de Datos, IA y ML, con el fin de:
- Comprender la distribución de salarios en el sector.
- Comparar rangos salariales según roles y niveles de experiencia.
- Identificar tendencias generales del mercado laboral en áreas de datos e inteligencia artificial.
- Facilitar la interpretación de los datos mediante visualizaciones claras y estructuradas.

---

## Herramientas utilizadas
- **Power BI Desktop**: herramienta principal para el modelado, análisis y visualización de datos.
- **Datos salariales**: dataset utilizado como fuente para el análisis (preprocesado dentro del entorno de Power BI).

## Estructura del análisis en Power BI
El reporte incluye, entre otros, los siguientes elementos:
- Visualización de salarios promedio y medianos.
- Comparación salarial por rol (Data Scientist, Machine Learning Engineer, entre otros).
- Análisis por nivel de experiencia.
- Gráficos comparativos y segmentadores para exploración interactiva.

---

## Alcance y limitaciones
- El análisis se basa únicamente en los datos disponibles en el dataset utilizado.
- Los resultados representan tendencias generales y no deben interpretarse como valores absolutos del mercado laboral.
- No se realiza modelado predictivo; el enfoque es descriptivo y exploratorio.
